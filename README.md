# 💬 ChatApp — Real-time Chat with WebSocket

## Gukora (How to Run)

### 1. Shyira ibikenewe (Install dependencies)
```bash
npm install
```

### 2. Tangira server (Start server)
```bash
node server.js
```

### 3. Fungura browser (Open browser)
```
http://localhost:3000
```

---

## Gukoresha (How to use)

1. Umuntu wa mbere afungura `http://localhost:3000`, yinjira izina rye no room (urugero: `general`)
2. Umuntu wa kabiri afungura browser inshya (cyangwa undi computer), akora igihe kimwe
3. Bombi bavuga mu gihe nyacyo! ✅

### Abantu benshi ku network imwe
Shaka IP ya computer yawe (urugero `192.168.1.5`), undi muntu yifashishe:
```
http://192.168.1.5:3000
```

---

## Features
- ✅ Real-time messaging (WebSocket)
- ✅ Rooms — ushobora gukora rooms nyinshi
- ✅ Typing indicator (ugaragara uri kwandika)
- ✅ Online users list
- ✅ Auto-reconnect niba internet yagiye
- ✅ Mobile friendly

## Stack
- **Backend**: Node.js + `ws` (WebSocket library)
- **Frontend**: HTML + CSS + Vanilla JS
- **Protocol**: WebSocket (ws://)
